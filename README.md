## 🚀 快速部署

### 前置要求
1. 微信小程序账号（30元认证费）
2. 微信开发者工具

### 第一步：下载后端服务

```bash
# 克隆项目
git clone https://github.com/370205504-cmyk/yushan-ai-cashier-assistant.git
cd yushan-ai-cashier-assistant

# 安装依赖
npm install

# 启动服务
node lambda/server.js
```

### 第二步：导入微信小程序

1. 打开微信开发者工具
2. 导入 `miniprogram` 目录
3. 配置云开发环境ID
4. 上传云函数
5. 创建数据库集合

详细步骤见：[微信小程序部署指南](miniprogram/README.md)

## 💡 使用场景

### 场景一：顾客扫码点餐
1. 商家在后厨打印桌码（每个桌子贴一张）
2. 顾客入座后微信扫码
3. 扫码后自动打开点餐小程序
4. 选择菜品 → 提交订单 → 微信支付
5. 订单自动打印小票到后厨

### 场景二：会员营销
1. 顾客首次消费后自动成为会员
2. 会员可充值享受免单机会
3. 商家可设置充值赠送活动

### 场景三：AI智能客服
1. 顾客在小程序中输入问题
2. AI自动回答（营业时间、WiFi、菜品咨询等）
