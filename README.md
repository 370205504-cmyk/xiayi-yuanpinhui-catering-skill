## 🚀 快速开始（5分钟上手）

### 方式一：商家使用（电脑）

```bash
# 1. 运行打包脚本
build.bat

# 2. 解压生成的安装包
# build/yushan-ai-cashier-assistant-v5.0.1-windows.zip

# 3. 解压到 D:\YushanAI

# 4. 双击【启动服务.bat】

# 5. 浏览器访问 http://localhost:3000
```

### 方式二：顾客使用（手机）

```
1. 进店
   ↓
2. 扫桌上的二维码
   ↓
3. 手机打开微信小程序
   ↓
4. 浏览菜单、点餐
   ↓
5. 下单支付
   ↓
6. 厨房收到订单，出餐
   ↓
7. 用餐
   ↓
8. 结账离店
```

### 配置方式
在项目根目录的 `.env` 文件中配置：
```env

QWEN_API_KEY=你的API密钥
QWEN_MODEL=qwen-turbo
```
