# 雨姗AI收银助手 v5.0.0

雨姗AI收银助手 - 市面所有收银系统通用AI智能增强助手。不用换收银，不用重新录菜品，不用培训员工，5分钟上线AI点餐、语音点餐、企业微信机器人点餐。

## 🎯 核心目标

### 第一阶段：AI 全自动读懂商家现有收银系统，不用商家手动录一条数据

| 序号 | 技能点 | 实现逻辑 | 商家价值 |
|------|--------|----------|----------|
| 1️⃣ | **自动识别本地收银系统** | 扫描电脑进程/注册表/数据库端口，自动匹配主流收银 | 商家不用选、不用配，打开软件自动对接 |
| 2️⃣ | **全量数据自动同步** | 只读模式拉取原有收银：菜品/价格/套餐/库存/桌台/会员/历史订单 | 不用手动录菜品、建会员，10分钟完成初始化 |
| 3️⃣ | **实时数据增量监听** | 监听收银数据库/接口，实时同步新订单/退单/改价/结账/库存变动 | AI 永远和收银数据一致，不会出现菜品下架还能点的问题 |
| 4️⃣ | **小票打印逆向解析** | 监听收银机网口/串口打印机，解析小票内容生成结构化订单 | 适配所有无API、无数据库权限的杂牌收银机，100%覆盖 |
| 5️⃣ | **数据自动校验纠错** | AI 自动识别重复菜品、错误价格、异常库存，提醒商家修正 | 避免同步数据出错导致的错单、漏单 |

### 第二阶段：AI 虚拟前台（核心必做，省人工）

**核心目标：替代80%前台接待/点单/咨询工作，原有收银只负责最后收钱**

| 序号 | 技能点 | 实现逻辑 | 商家价值 |
|------|--------|----------|----------|
| 1️⃣ | **无关键词自然语义点餐** | 大模型意图识别，支持大白话、模糊表达、口语化点餐 | 顾客说"来个两人份辣的套餐""上次那个汤再来一份"都能听懂 |
| 2️⃣ | **主动式智能推荐** | 结合历史销量、库存、顾客口味、当前活动，主动推荐菜品/套餐/凑单 | 提高客单价10%-15%，不用收银员推销 |
| 3️⃣ | **全场景自动答疑** | 覆盖营业时间/包间/停车/打包/辣度/忌口/优惠等200+常见问题 | 不用员工回微信、接电话答疑，AI秒回 |
| 4️⃣ | **多模态交互支持** | 支持文字、语音、图片识别（拍菜单点菜） | 老人、小孩都能轻松用，不用教 |
| 5️⃣ | **异常情况自动转人工** | AI识别回答不了的问题/复杂订单，自动转给前台人工处理 | 保证服务质量，不会出现AI答非所问的尴尬 |

### 第三阶段：Windows 绿色软件包（v5.0.0 新增）

**核心目标：打包成开箱即用的Windows绿色软件包，商家5分钟上线**

| 序号 | 模块名称 | 验收标准 |
|------|----------|----------|
| 1️⃣ | **一键配置向导** | 引导商家完成收银对接、二维码生成、打印机配置 | 商家从启动到上线不超过5分钟 |
| 2️⃣ | **二维码自动生成** | 通用店铺码、桌码自动生成，一键打印 | 输入桌号数量自动生成所有桌码 |
| 3️⃣ | **绿色软件打包** | 集成Node.js、MySQL、Redis所有运行环境，打包成单文件 | 单文件ZIP包≤100MB，解压双击即可启动 |
| 4️⃣ | **性能优化** | 内存占用优化到≤500MB，后台静默运行 | 不影响原有收银系统运行速度 |
| 5️⃣ | **离线运行支持** | 核心功能（点餐、打单、同步收银）断网可用 | 断网时订单本地缓存，联网后自动同步 |
| 6️⃣ | **数据安全** | 本地数据加密存储，自动备份与恢复 | 所有数据不上传云端，支持一键备份恢复 |
| 7️⃣ | **最终文档** | 编写《商家快速上手手册》《常见问题解答》 | 零基础商家能看懂并独立操作 |

## ⚡ 快速开始（Windows用户）

### 一键启动
1. 下载 `yushan-ai-cashier-assistant-v5.0.0-windows.zip`
2. 解压到任意目录（推荐：D:\YushanAI）
3. 双击 `启动服务.bat`
4. 等待约10秒，浏览器会自动打开配置向导
5. 5分钟完成配置，开始使用！

### 管理后台
- 访问：http://localhost:3000/admin
- 初始账号：admin
- 初始密码：admin123

### 顾客端
- 访问：http://localhost:3000
- 或扫描桌码

## 功能特点

### 核心功能
- 多渠道点餐：语音点餐、文字聊天、Web/移动端点餐、AI Agent对话
- AI Agent适配：支持扣子/龙虾/Dify等AI平台，完整MCP标准工具集
- 商业服务查询：20+种自然语言查询（WiFi、停车、营业时间等）
- 智能排队：排队叫号、实时进度查询、取消排队
- 完整购物车：加菜、减菜、口味备注、多人点餐
- 订单生命周期：待确认→已接单→制作中→已出餐→已完成/已取消
- 支付集成：微信支付、支付宝、余额支付、扫码支付
- 会员系统：积分、充值、优惠券、会员等级
- 多租户支持：SaaS架构，支持多门店

### v4.2.0 新增（可插拔适配器）
- 可插拔适配器架构：所有收银系统对接都实现统一接口
- 美团收银/银豹/哗啦啦/思迅/科脉 五大主流收银系统适配
- 数据库直连适配器：支持MySQL/Access/SQL Server，自动识别表结构
- 打印旁路兜底适配器：监听网口/串口打印机，小票逆向解析，虚拟打印入单
- 自动扫描与一键对接：启动后自动扫描本地进程、数据库、打印机，自动匹配适配器
- 双向数据同步引擎：增量同步、冲突解决、数据校验、防错机制

### v4.3.0 新增（AI智能体）
- MCP工具扩展：新增收银对接相关的工具（拉取收银菜品、同步订单到收银、查询收银库存）
- 自然语义理解升级：支持大白话点餐（如"来个两人份辣的套餐"、"跟上次一样12点到店取"）
- 上下文记忆增强：记住顾客口味、历史订单、当前会话状态
- AI主动技能：主动迎宾、主动推荐菜品、主动提醒口味忌口、订单状态自动推送
- 企业微信机器人：基于扣子平台，实现加好友、私聊、语音点餐、订单通知
- 自动转人工机制：AI识别回答不了的问题，自动推送人工客服二维码
- AI经营简报：每日自动生成营收、客流、爆款菜品分析
- 智能推荐引擎：结合历史销量、顾客口味、当前活动，主动推荐菜品/套餐/凑单
- 全场景FAQ答疑系统：覆盖200+常见问题，自动答疑
- 多模态交互：支持文字、语音、图片识别（拍菜单点菜）

### v5.0.0 新增（Windows绿色版）
- 一键配置向导：5分钟完成配置
- 二维码自动生成：店铺码、桌码一键生成
- 绿色软件打包：解压即用，无需安装
- 性能优化：内存占用≤500MB
- 离线运行支持：断网可用
- 数据安全：本地加密，自动备份
- 完善文档：商家快速上手手册+FAQ
