## 整合实现详情

### ✅ 成本核算模块 (Recipe Costing)
**文件**: `lambda/services/cost-accounting.js`  
**API**: `/api/v1/cost-accounting/*`

**功能**:
- 菜品成本计算（原料+人工+ overhead）
- 菜单工程分析（明星/现金牛/问题/瘦狗）
- 盈利能力分析（毛利率、food cost率）
- 批量成本计算
- 食谱管理

**测试结果**:
```
宫保鸡丁：
  - 原料成本: 11.8 元
  - 人工成本: 6.25 元
  - 总成本: 21.05 元
```

## 整合注意事项

1. **技术栈差异**
   - Python/Flask → Node.js/Express
   - Java/SpringBoot → Node.js/Express
   - C#/WPF → Web技术

2. **数据库统一**
   - 所有模块使用SQLite
   - 保持与雨姗AI收银助手一致

3. **API设计**
   - 遵循RESTful规范
   - 与现有API保持一致
