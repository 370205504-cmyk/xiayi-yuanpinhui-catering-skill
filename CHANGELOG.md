## [1.0.0] - 2024-05-11

### 🎉 初始版本发布

#### Added
- ✅ **菜品推荐功能**
  - 支持按菜系筛选（川菜、粤菜、湘菜等）
  - 支持按食材筛选（鸡肉、猪肉、豆腐等）
  - 支持按口味筛选（辣、清淡、酸甜等）
  
- ✅ **菜单生成功能**
  - 支持生成早餐、午餐、晚餐菜单
  - 支持生成一日三餐完整菜单
  - 支持按人数定制菜单

- ✅ **菜谱查询功能**
  - 提供 20 道经典中餐的详细做法
  - 包含食材清单、烹饪步骤、时间、难度、热量
  - 支持查询上次推荐的菜品

- ✅ **随机推荐功能**
  - 随机选择一道菜品
  - 解决"今天吃什么"的选择困难

- ✅ **多语言支持**
  - 中文 (zh-CN) 完整支持
  - 英文 (en-US) 基础支持

- ✅ **技术实现**
  - 基于 ASK SDK v2 for Node.js
  - AWS Lambda 部署支持
  - CloudFormation 基础设施模板
  - 完整的交互模型定义

## 版本说明

### 版本号格式

`MAJOR.MINOR.PATCH`

- **MAJOR**: 不兼容的 API 更改
- **MINOR**: 向下兼容的功能添加
- **PATCH**: 向下兼容的问题修复

### 标签说明

- 🎉 重大更新
- ✨ 新功能
- 🐛 Bug 修复
- 📝 文档更新
- 🔧 工具/配置
- ⚡ 性能优化
- ♻️ 代码重构
