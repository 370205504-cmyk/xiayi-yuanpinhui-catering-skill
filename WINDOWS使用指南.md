# Windows本地版使用指南

## 🚀 快速开始

### 第一步：安装 Node.js（如果没有）

1. 打开网址：https://nodejs.org/
2. 下载 **LTS** 版本（推荐）
3. 双击安装，一路点"下一步"
4. 安装完成后，**重新打开这个文件夹**

### 第二步：运行演示

双击运行 **`start-demo.bat`**

如果提示缺少依赖，先运行 **`install.bat`** 安装依赖

### 第三步：打开浏览器体验

服务启动后会自动打开浏览器，你也可以手动访问：

| 地址 | 说明 |
|------|------|
| http://localhost:3000 | 👤 顾客点餐页面 |
| http://localhost:3000/admin | 👨‍💼 管理后台 |
| http://localhost:3000/mobile | 📱 移动端预览 |

## ⚠️ 注意事项

### 这是演示版本
- 数据存储在内存中，**重启后会清空**
- 没有连接真实支付（微信/支付宝）
- 适合**体验功能**，不适合正式营业

### 正式使用需要
1. 云服务器（推荐腾讯云/阿里云）
2. 域名和SSL证书
3. 微信支付商户号
4. 支付宝商户号

## 🔧 常见问题

**Q: 提示"不是内部或外部命令"**
A: 请先安装 Node.js

**Q: 端口被占用（3000）**
A: 打开 `start-demo.bat`，把 `3000` 改成 `3001` 或其他端口

**Q: 浏览器显示空白**
A: 确保点击 `start-demo.bat` 后等待出现 "启动成功" 提示
